We can also use a function to describe this curve. Such a function is a Probability Density Function (PDF). In this case, this curve can be described in a normal distribution function.

Normal Distribution

The PDF of the normal distribution (sometimes called the Gaussian distribution) is:

\[f(x)=\frac{1}{\sigma\sqrt{2 \pi}} e^{-\frac{1}{2}(\frac{x-\mu}{\sigma})^{2} }\] In the case of the CO2 data:

mu <-  mean(co2)
mu
## [1] 396.0016
n <- length(co2)
sigma <- sd(co2) * sqrt((n-1)/n)
sigma
## [1] 2.579715
nd <- data.frame(x = seq(385, 406, 0.1))
nd$fx <- 1 /  (sigma * sqrt(2 * 3.14)) * 
  exp(-(nd$x - mu)^2 / (2 * sigma ^ 2 ))

plot(nd$x, nd$fx, type = "l")

f(x) is the density shown in previous graphs. we need to depend on concepts introduced in basic calculus to describe these distributions.

\[P(x_1 < x < x_2) = \int_{x_1}^{x_2} f(x)\text d x = \int_{x_1}^{x_2}\frac{1}{\sigma\sqrt{2 \pi}} e^{-\frac{1}{2}(\frac{x-\mu}{\sigma})^{2} } \text d x\]

For example, how many observations appeared between 394.0625 and 396.0625?

p395 <- pnorm(395, mean = mu, sd = sigma)
p397 <- pnorm(397, mean = mu, sd = sigma)
p397 - p395
## [1] 0.3017173
p394.5 <- pnorm(394.5, mean = mu, sd = sigma)
p396.5 <- pnorm(396.5, mean = mu, sd = sigma)
p396.5 - p394.5
## [1] 0.2963413
p394.0625 <- pnorm(c(394.0625), mean = mu, sd = sigma)
p396.0625 <- pnorm(c(396.0625), mean = mu, sd = sigma)
p396.0625 - p394.0625
## [1] 0.2832912

The PDF of the normal distribution has the following features:

  • It is bell shaped and symmetrical about its mean \(\mu\).
  • The mean, median and mode are all equal.
  • The total area under the curve is one unit.
  • Approximately 68%, 95% and 99.7% of the area is within 1, 2 and 3 standard deviations from the mean.
  • It is completely determined by the parameters \(\mu\) and \(\sigma\).
  • Different values of \(\mu\) shift the graph along the horizontal axis. Different values of \(\sigma\) alter the degree of flatness and peakedness of the curve. This is the reason why \(\mu\) is known as the location parameter and \(\sigma\) as the shape parameter.
  • The probability between two values \(a\) and \(b\) may be expressed as \(P(a<X<b) = \int_a^bf(x)\text dx\), which confirms that \(P(X=c) = \int_c^cf(x)\text dx = 0\).

How to read the \(z\) table:

  • The probability for \(z < -1\) is 0.1587.
  • The probability for \(z < 1.53\) is 0.9370.
  • The probability for \(-1 < z < 1.53\) is \(0.9370 - 0.1587 = 0.7783\).

z distribution

The PDF of the standard normal distribution has the following features:

  • It is bell shaped and symmetrical about its mean \(\mu\).
  • The mean, median and mode are all 0.
  • The total area under the curve is one unit.
  • Approximately 68%, 95% and 99.7% of the area is within \(\pm 1\), \(\pm 2\) and \(\pm 3\), respectively.
  • The probability between two values \(a\) and \(b\) (\(P(a<X<b)\)) in a normal distribution equals to the probability between \(\frac{a-\mu}{\sigma}\) and \(\frac{b-\mu}{\sigma}\) in a standard normal distribution.

If a quantitative variable \(x\) has a normal distribution with the mean of \(\mu\) and standard deviation of \(\sigma\), then we can create a new variable \(z_\mathrm {score}=(x-\mu)/\sigma\), and \(z_\mathrm {score}\) has a standard normal distribution.

For example, we convert a normal distribution into a standard normal distribution for the height data:

z_score <- (h - mu) /sigma

Convert a normal distribution into a standard normal distribution

If we want to know the percentage of the frequency in the range between two given values of the height (\(h_1\) and \(h_2\)), we can convert the height values into \(z\) (\(z_1\) and \(z_2\)), and look them up in the \(z\) table.

Demo:

The height in the ENV203 students data.

Estimate the percentage of students between 162 and 172 cm with the \(z\) table. 
mu <- mean(h)
n <- length(h)
sigma <- sd(h) * sqrt((n-1) /n)
(162 - mu)/sigma
## [1] -0.8698527
# look up the z table. -0.87 ---- 0.1922
(172 - mu)/sigma
## [1] 0.5286243
# look up the z table. 0.53 ---- 0.7019
0.7019 - 0.1922
## [1] 0.5097

The probability of \(\mu - z_0\sigma < x < \mu + z_0\sigma\) (or \(\mu \pm z_0\sigma\)) in the normal distribution of \(x\) equals to the probability of \(-z_0 < z < z_0\) in the standard normal distribution of \(z\). In this way, we can get the percentage (possibility) of any given range in a normal distribution by simple arithmetic calculation and the \(z\) table.
